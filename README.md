# Mini-ATM
Mini ATM project using oops concept 
Project Overview
This mini project implements a simple ATM Management System using Python Object-Oriented Programming (OOPS) concepts.
Real-World ATM Operations Simulated
The system simulates the following ATM operations:
Checking account balance
Depositing money
Withdrawing money
Viewing account details
Safely exiting the system
Program Behavior
The program is menu-driven and runs continuously until the user chooses the Exit option.
Objectives of the Project
To understand real-world system modeling using OOPS
To apply abstraction, encapsulation, inheritance, and polymorphism
To build a clean, beginner-friendly console application
To improve logical thinking using loops and conditional statements
